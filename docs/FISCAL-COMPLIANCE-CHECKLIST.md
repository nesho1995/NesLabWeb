# Checklist Fiscal y Compliance (Honduras)

> Este checklist debe validarse y firmarse con contador/fiscalista local antes de salida productiva.

## 1) Configuracion CAI y facturacion
- [ ] CAI vigente configurado correctamente.
- [ ] Rango de facturacion cargado y correlativo validado.
- [ ] Fecha limite CAI correcta.
- [ ] Politica de documentos provisionales definida.

## 2) Operacion offline y regularizacion
- [ ] Flujo offline documentado para personal.
- [ ] Regularizacion al reconectar probada end-to-end.
- [ ] Bitacora de regularizaciones exportable y consistente.
- [ ] Ordenes muestran estado fiscal (pendiente/regularizada).

## 3) Reportes y trazabilidad
- [ ] Reportes financieros revisados por administracion.
- [ ] Filtros por fecha/metodo funcionan.
- [ ] Bitacora de acciones criticas habilitada y revisable.

## 4) Roles y permisos
- [ ] Mapa rol-permisos revisado por direccion.
- [ ] Ningun rol operativo tiene permisos admin innecesarios.
- [ ] Altas/bajas de usuarios con control dual (solicitante/aprobador).

## 5) Acta de validacion final
- Responsable tecnico:
- Contador/fiscalista:
- Fecha:
- Observaciones:
- Aprobado para produccion: SI / NO
